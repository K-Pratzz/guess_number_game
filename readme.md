This is a simple number guessing game built using basic HTML, CSS, and JavaScript.  
The user enters a guess, and the program checks whether it matches the secret number.  
The game gives feedback (Too high / Too low / Correct).

 Installation & Usage
1. Download the project files (index.html, style.css, script.js).
2. Keep all files in the same folder.
3. Open *index.html* in any web browser.
4. Start guessing numbers!
